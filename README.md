# action-repo

This is a dummy GitHub repository used to trigger webhook events for the TechStaX Developer Assessment.

## Used For:
- Push events
- Pull request events
- Merge events

These events are sent to a Flask backend (`webhook-repo`) and displayed in a live UI.
